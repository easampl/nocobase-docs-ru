# Продвинутые возможности

## Введение

ИИ сотрудников можно привязывать к страницам или блокам. После привязки вы сможете настроить задачи, специфичные для текущего бизнес-процесса, что позволит пользователям быстро использовать ИИ сотрудника для их выполнения прямо на странице или в блоке.

## Привязка ИИ сотрудника к странице

Когда страница переходит в режим редактирования UI, в правом нижнем углу рядом с кнопкой быстрого вызова ИИ сотрудника появится значок «+». Наведите курсор на этот значок, чтобы открыть список ИИ сотрудников. Выберите нужного ИИ сотрудника, и он будет привязан к текущей странице.

![20251022134656](https://static-docs.nocobase.com/20251022134656.png)

После завершения привязки при каждом входе на страницу в правом нижнем углу будет отображаться ИИ сотрудник, привязанный к текущей странице.

![20251022134903](https://static-docs.nocobase.com/20251022134903.png)

## Привязка ИИ сотрудника к блоку

Когда страница переходит в режим редактирования UI, для блока, поддерживающего настройку «Действия» (`Actions`), выберите пункт «ИИ сотрудники» (`AI employees`) в разделе «Действия» и укажите ИИ сотрудника, которого нужно привязать к текущему блоку.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

После завершения привязки при каждом входе на страницу в области «Действия» (`Actions`) блока будет отображаться ИИ сотрудник, привязанный к текущему блоку.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## Настройка задач

Когда страница в режиме редактирования UI, наведите курсор на значок ИИ сотрудника, привязанного к странице или блоку. Появится кнопка меню. Выберите «Редактировать задачи» (`Edit tasks`), чтобы открыть страницу настройки задач.

![20251022135710](https://static-docs.nocobase.com/20251022135710.png)

На странице настройки задач добавьте несколько задач для текущего ИИ сотрудника.

Каждая вкладка — отдельная задача. Нажмите значок «+» рядом с ней, чтобы добавить новую задачу.

![20251022140058](https://static-docs.nocobase.com/20251022140058.png)

Форма настройки задач:

- В поле «Заголовок» (`Title`) введите заголовок задачи. Кратко опишите её содержание. Этот заголовок будет отображаться в списке задач ИИ сотрудника.
- В поле «Описание задачи» (`Background`) укажите основное содержание задачи. Этот текст будет использоваться в качестве системного промпта при общении с ИИ сотрудником.
- В поле «Сообщение пользователя по умолчанию» (`Default user message`) введите текст, который будет автоматически подставляться в поле ввода пользователя после выбора задачи.
- В блоке «Рабочий контекст» (`Work context`) выберите информацию о контексте приложения по умолчанию, которая будет отправлена ИИ сотруднику. Эта операция аналогична настройке в диалоговом окне.
- В поле выбора «Навыки» (`Skills`) отображаются доступные навыки текущего ИИ сотрудника. Отключите отдельные навыки, чтобы ИИ сотрудник не использовал их при выполнении этой задачи.
- Флажок «Автоматически отправлять сообщение пользователя по умолчанию» (`Send default user message automatically`) определяет, будет ли сообщение по умолчанию отправляться сразу после нажатия кнопки выполнения задачи.

![20251022140805](https://static-docs.nocobase.com/20251022140805.png)

## Список задач

После настройки задач для ИИ сотрудника они будут отображаться во всплывающем окне профиля ИИ сотрудника и в приветственном сообщении перед началом беседы. Нажмите на задачу, чтобы выполнить её.
