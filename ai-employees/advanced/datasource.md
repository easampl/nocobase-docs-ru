# Расширенные возможности

## Введение

В плагине ИИ сотрудников вы можете настроить источники данных и предварительно задать запросы к коллекциям. Эти запросы отправляются в качестве контекста при общении с ИИ сотрудниками, которые затем отвечают на основе полученных результатов.

## Настройка источников данных

Перейдите на страницу настроек плагина ИИ сотрудников, нажмите вкладку «Источник данных» (`Data source`), чтобы попасть на страницу управления источниками данных ИИ сотрудников.

![20251022103526](https://static-docs.nocobase.com/20251022103526.png)

Нажмите кнопку «Добавить источник данных» (`Add data source`), чтобы перейти на страницу создания источника данных.

Шаг 1: Введите основную информацию о `коллекции`:
- В поле «Название» (`Title`) введите легко запоминающееся название для источника данных;
- В поле «Коллекция» (`Collection`) выберите источник данных и коллекцию, которые вы хотите использовать;
- В поле «Описание» (`Description`) введите описание источника данных;
- В поле «Лимит» (`Limit`) укажите ограничение на количество записей в запросе к источнику данных, чтобы избежать возврата слишком большого объема данных, который может превысить контекст диалога с ИИ сотрудниками.

![20251022103935](https://static-docs.nocobase.com/20251022103935.png)

Шаг 2: Выберите поля для запроса:

В списке «Поля» (`Fields`) отметьте поля, которые вы хотите запросить.

![20251022104516](https://static-docs.nocobase.com/20251022104516.png)

Шаг 3: Установите условия запроса:

![20251022104635](https://static-docs.nocobase.com/20251022104635.png)

Шаг 4: Установите условия сортировки:

![20251022104722](https://static-docs.nocobase.com/20251022104722.png)

Перед сохранением источника данных предварительно просмотрите результаты запроса к источнику данных.

![20251022105012](https://static-docs.nocobase.com/20251022105012.png)

## Отправка источников данных в диалогах

В диалоговом окне ИИ сотрудников нажмите кнопку «Добавить рабочий контекст» (`Add work context`) в левом нижнем углу, выберите «Источник данных» (`Data source`), и вы увидите только что добавленный источник данных.

![20251022105240](https://static-docs.nocobase.com/20251022105240.png)

Отметьте источник данных, который вы хотите отправить; выбранный источник данных будет прикреплен к диалоговому окну.

![20251022105401](https://static-docs.nocobase.com/20251022105401.png)

После ввода вопроса, как и при отправке обычного сообщения, нажмите кнопку отправки, и ИИ сотрудники ответят на основе источника данных.

Источник данных также появится в списке сообщений.

![20251022105611](https://static-docs.nocobase.com/20251022105611.png)

## Примечания

Источник данных автоматически отфильтрует информацию на основе прав доступа ACL текущего пользователя, показывая только те данные, к которым у пользователя есть доступ.
