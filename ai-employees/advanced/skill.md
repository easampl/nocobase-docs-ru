# Расширенные возможности

## Введение

Современные языковые модели (LLM) могут использовать внешние инструменты. Плагин «ИИ сотрудники» включает набор встроенных средств, доступных LLM.

Навыки, которые вы настраиваете на странице параметров ИИ сотрудников, представляют собой инструменты, доступные LLM.

![20251022142348](https://static-docs.nocobase.com/20251022142348.png)

## Настройка навыков

Перейдите на страницу конфигурации плагина ИИ сотрудников и нажмите вкладку «ИИ сотрудники» (`AI employees`), чтобы открыть страницу управления.

Выберите ИИ сотрудника, для которого нужно настроить навыки, и нажмите кнопку «Редактировать» (`Edit`), чтобы открыть страницу редактирования.

На вкладке «Навыки» (`Skills`) нажмите кнопку «Добавить навык» (`Add Skill`), чтобы создать новый навык для выбранного ИИ сотрудника.

![20251022145748](https://static-docs.nocobase.com/20251022145748.png)

## Обзор навыков

### Фронтенд (`Frontend`)

Группа «Фронтенд» (`Frontend`) позволяет взаимодействовать с компонентами пользовательского интерфейса.

- Навык «Заполнение формы» (`Form filler`) заполняет сгенерированные данные в указанную пользователем форму.

![20251022145954](https://static-docs.nocobase.com/20251022145954.png)

### Моделирование данных

Группа «Моделирование данных» предоставляет возможность вызывать внутренние API NocoBase для моделирования данных.

- «Маршрутизатор намерений» (`Intent Router`) определяет, нужно ли изменить структуру **коллекции** или создать новую.
- «Получение названий коллекций» (`Get collection names`) возвращает названия всех существующих **коллекций** в системе.
- «Получение метаданных коллекции» (`Get collection metadata`) возвращает информацию о структуре указанной **коллекции**.
- «Определение коллекций» (`Define collections`) создаёт новые **коллекции** в системе.

![20251022150441](https://static-docs.nocobase.com/20251022150441.png)

### Вызов рабочего процесса (`Workflow caller`)

Навык «Вызов рабочего процесса» (`Workflow caller`) выполняет настроенные **рабочие процессы** (Workflow). **Рабочие процессы**, созданные в **плагине рабочих процессов** с типом триггера (`Trigger type`) «событие ИИ сотрудника» (`AI employee event`), становятся здесь доступными как навыки.

![20251022153320](https://static-docs.nocobase.com/20251022153320.png)

### Редактор кода

Навыки, входящие в группу «Редактор кода», предназначены для взаимодействия с редактором кода.

- «Получение списка фрагментов кода» (`Get code snippet list`) возвращает список предустановленных фрагментов кода.
- «Получение содержимого фрагмента кода» (`Get code snippet content`) возвращает содержимое указанного фрагмента кода.

![20251022153811](https://static-docs.nocobase.com/20251022153811.png)

### Прочее

- Навык «Генератор диаграмм» (`Chart generator`) генерирует диаграммы и выводит их в диалоге.

![20251022154141](https://static-docs.nocobase.com/20251022154141.png)
