# Работа с ИИ-сотрудниками

После создания и включения ИИ-сотрудников вы можете сотрудничать с ними на страницах.

## Точки входа

Есть два основных варианта входа:

1. **Основной вход в правом нижнем углу**: откройте панель чата ИИ в правом нижнем углу бизнес-страниц. Подходит для общих вопросов и ответов, а также работы с несколькими блоками.
2. **Вход через действие блока**: в блоках, которые поддерживают `Действия`, вход выполняется через `Действия -> ИИ-сотрудники`. Подходит для задач в текущем блоке (например, сценарии JS-блок).

## Основные операции в чате

Панель чата поддерживает типовые действия: отправку сообщений, загрузку вложений, просмотр истории, создание новых чатов и редактирование системных описаний.

## Переключение в чате

Внизу области ввода можно выполнить переключение напрямую:

- `Переключатель ИИ-сотрудника`: переключает текущего сотрудника.
- `Переключатель модели`: переключает модель текущей сессии.

Предпочтения моделей сохраняются отдельно для каждого сотрудника и восстанавливаются при следующем запуске.
