# Использование навыков

Навыки (инструменты) определяют, что ИИ-сотрудник может делать.

## Структура навыков

Страница навыков (Skills) разделена на три секции:

1. `Общие навыки`: общие для всех ИИ-сотрудников, только для чтения.
2. `Навыки, специфичные для сотрудника`: специфичны для текущего сотрудника, обычно только для чтения.
3. `Пользовательские навыки`: пользовательские навыки, которые можно добавлять/удалять и настраивать с правами по умолчанию.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## Модель прав

Права унифицированы как:

- `Спросить`: запрос подтверждения перед вызовом.
- `Разрешить`: разрешить прямой вызов.

Рекомендация: используйте `Спросить` по умолчанию для навыков, изменяющих данные.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## Добавление и поддержка

Используйте `Добавить навык` в `Пользовательских навыках`, чтобы добавлять навыки и настраивать права в зависимости от уровня риска.
